Automação Industrial com CLP | Influx DB | Grafana | N8N

publicado em 05/06/2025
Este é um projeto que estou desenvolvendo atualmente para otimizar a linha de produção, custos com funcionarios que não são mais
necessarios, e melhorar a qualidade do produto final.

O projeto consiste em integrar um CLP (Controlador Lógico Programável) com o Influx DB e o Grafana para monitoramento e visualização dos dados.
também estou utilizando o N8N para automatizar o processo de coleta e envio dos dados.

O objetivo é coletar os dados de produção em tempo real, armazená-los no Influx DB e visualizá-los no Grafana.
mas também estou utilizando o N8N para enviar emails de solicitação de materiais , ensumos e outros materiais necessários para a produção.

O projeto é uma ótima oportunidade para aprender mais sobre automação industrial e como integrar diferentes tecnologias para otimizar processos.

Cinetop - o que é ?

publicado em 03/09/2025
Cinetop é uma aplicação web que criei para resolver os seguintes problemas: em quais provedores de streaming encontrar os principais
lançamentos de filmes
como Netflix, Amazon Prime Video, Disney+ e outros.
aproveite para baixar o aplicativo no seu celular e ter acesso a todos os filmes e séries disponíveis em um só lugar.

Com o Cinetop, você pode pesquisar por filmes e séries, verificar quais estão disponíveis em cada serviço de streaming e até mesmo ver as últimas novidades.

Além disso, você pode encontrar informações sobre os filmes, como sinopse, elenco e avaliações, tudo em um só lugar.
Existe uma opção de integrada de um assistente de inteligencia artificial que funciona como um anfitrião de uma especie de cinematéca.
